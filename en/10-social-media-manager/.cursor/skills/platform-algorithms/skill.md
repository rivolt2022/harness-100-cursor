# Platform Algorithms — Platform Algorithm Methodology

Specialized algorithm knowledge used by the sns-strategist and visual-planner agents when designing content strategy and visuals.

## Why Understanding Algorithms Matters

Even great content can be **shown to only 10% of followers** by the algorithm. The algorithm is not the enemy — it is a **tool to be leveraged**.

## Instagram Algorithm (2025 Baseline)

### Feed Ranking Signals

| Signal | Weight | Description |
|--------|--------|-------------|
| **Relationship** | 30% | Frequency of prior interactions (likes, comments, DMs, tags) |
| **Interest** | 25% | History of consuming similar content |
| **Timeliness** | 20% | Time elapsed since posting (recent favored) |
| **Session Frequency** | 15% | More diverse content shown to frequent users |
| **Following Count** | 10% | More follows = lower individual content exposure probability |

### Algorithm Differences by Format

| Format | Key Metric | Optimization Strategy |
|--------|-----------|----------------------|
| **Feed Post** | Save > Share > Comment > Like | Information-rich content worth saving |
| **Reels** | Completion Rate > Share > Replay | Hook within 3 sec, 15-30 sec optimal, loop structure |
| **Carousel** | Swipe Rate > Save > Share | Hook on first slide, CTA on last |
| **Stories** | Reply > Poll/Quiz Participation > Tap Speed | Interactive stickers, questions |

### Instagram Golden Hours

| Audience | Optimal Posting Time |
|---------|---------------------|
| General | Tue-Fri 11:00-13:00, 19:00-21:00 |
| B2B | Tue-Thu 08:00-09:00 |
| Weekend Content | Sat 10:00-12:00 |

## TikTok Algorithm

### For You Page Ranking

| Signal | Weight | Description |
|--------|--------|-------------|
| **Completion Rate** | 35% | Watched to the end (most important) |
| **Re-watch** | 20% | Repeated viewing of same video |
| **Share** | 15% | DM/external sharing |
| **Comments** | 15% | Comment writing and length |
| **Profile View** | 10% | Visited profile after watching |
| **Follow** | 5% | Followed after watching |

### TikTok Optimization Essentials

1. **First 1 second is everything**: Visual/text hook to stop scrolling
2. **15-30 seconds optimal**: Maximize completion rate (longer videos reduce completion)
3. **Loop structure**: End flowing into beginning encourages re-watching
4. **Trending sounds**: Using popular music/audio boosts exposure
5. **Text overlay**: Subtitles essential for muted viewers

## LinkedIn Algorithm

### Content Classification and Reach

| Classification | Reach | Characteristics |
|---------------|-------|-----------------|
| **Native Text** | Highest | Text only, no external links |
| **Image Post** | High | Includes 1-3 images |
| **Document/PDF** | High | Slide-format document upload |
| **Video** | Medium | Native upload only (no YouTube links) |
| **External Link** | Low | Including external URLs reduces reach 40-50% |

### LinkedIn Golden Hour

The **first hour's reactions** after posting determine overall reach:
- 10+ comments in the first hour -> Algorithm boost
- **Strategy**: After posting, request opinions from relevant connections via DM

## X (Twitter) Algorithm

### For You Feed Signals

| Signal | Effect |
|--------|--------|
| **Retweet + Quote** | Highest weight — virality signal |
| **Reply Threads** | Conversation creation = content value |
| **Bookmark** | Signal strengthened since 2023 |
| **Dwell Time** | Content read at length is favored |
| **Image/Video** | 1.5-2x reach vs text-only |
| **External Link** | 30-50% reach decrease (separate to self-reply) |

# LinkedIn SEO — LinkedIn Profile Search Optimization Tool

A specialized skill that enhances the profile-optimizer agent's LinkedIn optimization capabilities.

## Target Agents

- **profile-optimizer** — LinkedIn profile optimization, keyword strategy

## Understanding the LinkedIn Search Algorithm

### Search Ranking Factors

| Factor | Weight | Optimization Method |
|--------|--------|-------------------|
| Keyword Matching | High | Strategic placement of target keywords |
| Profile Completeness | High | Achieve All-Star level |
| Connection Network | Medium | 500+ connections |
| Activity Level | Medium | 1-2 posts/comments per week |
| Recommendations/Endorsements | Medium | 5+ recommendations, skill endorsements |

### Keyword Placement Priority (by Search Weight)

```
1. Headline — Highest weight
2. Current Job Title
3. About (Summary) section
4. Experience descriptions
5. Skills list
6. Education
7. Recommendations
```

## Section-by-Section Optimization Guide

### Headline (120 characters)

**Formula:**
```
[Target Job Title] | [Core Expertise] | [Differentiating Keyword]
```

| Type | Good Example | Bad Example |
|------|-------------|-------------|
| Developer | "Senior Backend Engineer \| Java & Kotlin \| MSA & Cloud Architecture" | "Developer" |
| Marketer | "Growth Marketing Manager \| B2B SaaS \| SEO & Paid Acquisition" | "Marketing person" |
| PM | "Product Manager \| Fintech \| Data-driven Product Strategy" | "PM at Company" |

### About Section (2,600 characters)

**Structure:**
```
[Hook] — First 3 lines are critical (visible before "See more" click)
  "I am a [title] who has delivered [key achievement] in [field] for N years."

[Value Proposition] — Your unique value
  "My areas of expertise:"
  - [Expertise 1] + [Quantitative achievement]
  - [Expertise 2] + [Quantitative achievement]
  - [Expertise 3] + [Quantitative achievement]

[Story] — Career narrative (optional)
  "From how I entered [field] to my growth journey to date"

[Skills] — Keyword list
  "Core skills: [keyword1], [keyword2], [keyword3]..."

[CTA] — Contact prompt
  "If you'd like to discuss [topic], feel free to reach out."
```

### Experience Section

**Structure for each entry:**
```
[Title] at [Company]
[Duration]

[1-line role summary]

- [Keyword included] [Achievement 1] — [Quantitative result]
- [Keyword included] [Achievement 2] — [Quantitative result]
- [Keyword included] [Achievement 3] — [Quantitative result]

Technologies: [Related tech keywords]
```

### Skills Section

**Strategy:**
- Up to 50 skills can be registered
- **Top 3 skills** = The key skills displayed on the profile (set as target keywords)
- Match exactly with keywords extracted from JDs
- Request skill endorsements from colleagues

### Featured Section

| Content Type | Effect | Recommended |
|-------------|--------|-------------|
| Blog Posts | Proves expertise | 2-3 |
| Project Links | Proves ability | 1-2 |
| Talks/Conferences | Proves leadership | 1-2 |
| Media Interviews | Proves influence | When available |

## Recruiter Search Patterns

### Search Queries Used by Recruiters

```
Typical Search:
"Backend Engineer" AND "Java" AND "Spring Boot" AND "Seoul"

Advanced Search (LinkedIn Recruiter):
Title: "Senior" AND ("Backend" OR "Server")
Skills: "Java" AND "AWS"
Location: "Seoul"
Years of Experience: 5-10
```

### Checklist for Appearing in Searches

- [ ] Is the target job title exactly included in the Headline?
- [ ] Are core technical keywords registered in Skills?
- [ ] Is the current Location accurate?
- [ ] Is the Industry correctly set?
- [ ] Is the Open to Work setting activated? (Can be set to private)

## LinkedIn SSI (Social Selling Index) Optimization

| SSI Component | Weight | Optimization Actions |
|--------------|--------|---------------------|
| Profile Completeness | 25% | All-Star level, fill all sections |
| Finding the Right People | 25% | Expand connections in target industry |
| Sharing Insights | 25% | Post professional content 1-2 times per week |
| Building Relationships | 25% | Comments, messages, recommendation activity |

# Podcast Growth — 팟캐스트 성장 전략 방법론

distribution-manager 에이전트가 배포 패키지를 설계할 때 활용하는 청취자 확보 및 성장 전문 지식.

## 왜 이 스킬이 필요한가

좋은 콘텐츠는 필요조건이지 충분조건이 아니다. 전 세계 팟캐스트 500만+ 개. **발견되지 않으면 존재하지 않는 것**과 같다.

## 플랫폼별 메타데이터 최적화

### Apple Podcasts
| 필드 | 최적화 규칙 | 이유 |
|------|-----------|------|
| **제목** | 핵심 키워드를 앞부분에 (40자 이내) | 검색 알고리즘이 앞부분 우선 인덱싱 |
| **부제** | 제목에 없는 보조 키워드 (60자 이내) | 검색 범위 확장, 중복은 패널티 |
| **설명** | 첫 2문장에 핵심 + 청취자 이익 | 미리보기 160자만 노출 |
| **에피소드 제목** | "에피소드 N:" 형식 지양 | Apple이 명시적으로 비권장 |
| **카테고리** | 메인 1개 + 서브 2개 | 3개 차트 동시 노출 가능 |

### Spotify
| 필드 | 최적화 규칙 | 이유 |
|------|-----------|------|
| **에피소드 설명** | 타임스탬프 + 키워드 3~5회 자연 삽입 | 내부 검색 최적화 |
| **챕터 마커** | 세그먼트별 챕터 설정 | 부분 재생→전체 전환율 30% 향상 |
| **비디오 형식** | 정적 이미지라도 비디오로 업로드 | 비디오 팟캐스트 알고리즘 우선 노출 |

### YouTube Podcasts
| 필드 | 최적화 규칙 | 이유 |
|------|-----------|------|
| **썸네일** | 얼굴 클로즈업 + 3단어 이내 텍스트 | CTR 최적화 기본 |
| **제목** | 키워드 앞배치 + 감정 유발 단어 | 검색 + 추천 알고리즘 |
| **설명** | 첫 3줄 핵심 + 타임스탬프 + 관련 링크 | 체류 시간 및 채널 순환 |

## 에피소드 홍보 캘린더: 7일 플랜

| D-day | 채널 | 콘텐츠 | 목적 |
|-------|------|--------|------|
| **D-3** | 인스타 스토리 / X | 티저 클립 (15초) + "곧 공개" | 기대감 조성 |
| **D-1** | 뉴스레터 | 프리뷰 + 독점 비하인드 | 코어 팬 선행 알림 |
| **D-Day** | 모든 SNS | 에피소드 공개 + 인용구 카드 3장 | 최대 도달 |
| **D+1** | X (트위터) | 핵심 인사이트 스레드 (5~8트윗) | 바이럴 확산 |
| **D+2** | 릴스 / 틱톡 | 가장 재밌는 30초 클립 | 숏폼→풀 에피소드 전환 |
| **D+3** | 링크드인 | 전문적 인사이트 요약 + CTA | B2B 청취자 확보 |
| **D+5** | 커뮤니티 | 에피소드 관련 질문, 토론 유도 | 커뮤니티 활성화 |

## 홍보 카피 프레임워크: HIKE 공식

| 단계 | 설명 | 예시 |
|------|------|------|
| **H**ook | 스크롤을 멈추게 하는 첫 줄 | "AI가 당신의 직업을 대체하는 건 아닙니다. 더 무서운 일이 벌어지고 있어요." |
| **I**nsight | 에피소드에서만 들을 수 있는 발견 | "이번 에피소드에서 밝혀진 3가지 사실이 산업을 흔들 수 있습니다" |
| **K**ey moment | 가장 인상적인 순간 하나 | "게스트가 '저도 3번 실패했습니다'라고 말하는 순간..." |
| **E**ngage | 청취 유도 + 반응 요청 | "풀 에피소드 듣고 여러분 생각을 댓글로 [링크]" |

### 플랫폼별 HIKE 변형

| 플랫폼 | Hook 스타일 | 길이 | CTA |
|--------|-----------|------|-----|
| **X/트위터** | 숫자 또는 대비 | 280자 + 스레드 | "리트윗하고 에피소드 듣기" |
| **인스타그램** | 감정적 질문 | 캡션 150자 + 카드뉴스 | "프로필 링크에서 듣기" |
| **링크드인** | 전문적 문제 제기 | 3~4문단 (1,300자) | "팟캐스트 링크는 첫 댓글에" |
| **틱톡** | 충격/유머 | 자막 포함 30초 영상 | "풀 에피소드 바이오 링크" |

## 크로스 프로모션 전략

### 게스트 레버리지
1. 게스트에게 **즉시 공유 가능한 키트** 제공: 카드뉴스 3장 + 추천 카피 2종 + 링크
2. 게스트 에피소드 공개 24시간 내에 게스트를 태그하여 포스팅
3. 게스트의 콘텐츠(뉴스레터, 블로그)에 에피소드 언급 요청

### 팟캐스트 네트워크
1. 유사 주제 팟캐스트와 **에피소드 교환**: 서로의 에피소드에서 상대 추천
2. **공동 에피소드 제작**: 두 팟캐스트가 같은 주제를 각자의 관점에서
3. **프로모 스왑**: 30초 광고를 서로의 에피소드에 삽입

## 청취자 성장 지표

| 지표 | 건강한 수치 | 경고 수준 |
|------|-----------|----------|
| **다운로드 성장률** | 월 10%+ | 3개월 연속 0% 미만 |
| **에피소드 완주율** | 60%+ | 40% 미만 |
| **구독 전환율** | 5%+ | 1% 미만 |
| **SNS→팟캐스트 전환율** | 2~5% | 0.5% 미만 |

# Naming Methodology — 브랜드 네이밍 방법론

naming-specialist 에이전트가 브랜드명을 개발할 때 활용하는 네이밍 전문 지식.

## 왜 체계적 네이밍이 필요한가

좋은 이름은 **우연히 떠오르지 않는다**. "나이키"는 승리의 여신, "구글"은 10의 100승(구골). 성공한 브랜드명 뒤에는 반드시 전략적 사고가 있다.

## 네이밍 기법 12가지

### 1. 서술적 (Descriptive)
- **방법**: 제품/서비스를 직접 설명
- **장점**: 즉시 이해 가능
- **단점**: 차별화 어려움, 상표 등록 난이도 높음
- **예시**: YouTube (너의 TV), PayPal (지불 친구)

### 2. 연상적 (Evocative)
- **방법**: 브랜드 가치를 간접적으로 암시
- **장점**: 감정적 연결, 상표 등록 용이
- **단점**: 의미 전달에 시간 필요
- **예시**: Amazon (거대함, 무한), Apple (단순함, 신선)

### 3. 조어 (Coined / Invented)
- **방법**: 완전히 새로운 단어를 만듦
- **장점**: 고유성 최대, 상표 등록 최적
- **단점**: 의미 학습 비용
- **예시**: Kodak, Xerox, Spotify

### 4. 합성어 (Compound / Portmanteau)
- **방법**: 두 단어를 결합
- **장점**: 의미 전달 + 고유성
- **예시**: Facebook (얼굴+책), Instagram (즉석+전보), Netflix (인터넷+영화)

### 5. 두문자어 (Acronym)
- **방법**: 이니셜로 축약
- **장점**: 짧고 기억하기 쉬움
- **단점**: 의미 전달 약함, 이미 포화
- **예시**: IBM, BMW, LG

### 6. 지명/인명 (Eponymous)
- **방법**: 창업자 이름이나 지역명
- **장점**: 신뢰감, 유산
- **예시**: Samsung (삼성), Hyundai (현대), Chanel

### 7. 외래어 차용 (Foreign Language)
- **방법**: 다른 언어에서 의미 있는 단어 차용
- **장점**: 이국적, 의미 레이어 추가
- **예시**: Volvo (라틴어 "굴러가다"), Audi (라틴어 "듣다")

### 8. 의성어 (Onomatopoeia)
- **방법**: 소리를 이름으로
- **장점**: 감각적, 기억 용이
- **예시**: Zoom, Snap, Buzz

### 9. 은유 (Metaphor)
- **방법**: 다른 영역의 이미지를 차용
- **장점**: 풍부한 연상
- **예시**: Red Bull (에너지), Jaguar (속도/우아함)

### 10. 접두/접미 변형 (Prefix/Suffix)
- **방법**: 기존 단어에 접두/접미사 추가
- **장점**: 의미 확장 + 고유성
- **패턴**: -ify, -ly, -io, -eo, -ium, re-, un-, pre-
- **예시**: Spotify, Grammarly, Canva

### 11. 축약 변형 (Truncation)
- **방법**: 단어를 줄이거나 철자를 변형
- **장점**: 현대적, 간결
- **예시**: Tumblr (tumbler), Flickr (flicker), Lyft (lift)

### 12. 수학/기호 (Alphanumeric)
- **방법**: 숫자, 기호를 포함
- **장점**: 테크 느낌, 차별화
- **예시**: 7-Eleven, 3M, Y Combinator

## 네이밍 평가 매트릭스: SMILE 테스트

| 기준 | 질문 | 1~5점 |
|------|------|-------|
| **S**imple | 3초 안에 발음할 수 있는가? 철자를 물어볼 필요가 없는가? | |
| **M**emorable | 한 번 듣고 다음 날 기억할 수 있는가? | |
| **I**mage-evoking | 구체적 이미지나 감정이 떠오르는가? | |
| **L**egally available | 상표 등록 가능한가? 도메인은 확보 가능한가? | |
| **E**xpandable | 사업이 확장되어도 이름이 제한하지 않는가? | |

**합격 기준**: 총점 20점 이상 (25점 만점)

## 음운론적 분석

### 소리의 감정적 속성

| 음소 유형 | 감정적 연상 | 적합한 브랜드 유형 |
|----------|-----------|----------------|
| 파열음 (ㅂ,ㅃ,ㅍ,ㄷ,ㄸ,ㅌ,ㄱ,ㄲ,ㅋ) | 강렬, 힘, 결단 | 테크, 스포츠, 에너지 |
| 마찰음 (ㅅ,ㅆ,ㅎ,ㅈ,ㅉ,ㅊ) | 부드러움, 정교함 | 뷰티, 패션, 럭셔리 |
| 비음 (ㄴ,ㅁ,ㅇ) | 따뜻함, 안정 | 식품, 건강, 가족 |
| 유음 (ㄹ) | 유연, 우아 | 라이프스타일, 프리미엄 |
| 모음 'ㅏ,ㅗ' | 개방적, 밝음 | 긍정적 브랜드 |
| 모음 'ㅜ,ㅡ' | 깊음, 안정 | 전문적, 기술적 |

### 한국어 네이밍 특수 고려사항

- **초성 법칙**: 한국어에서 ㄹ로 시작하는 고유어는 없다 → 외래어 느낌
- **4글자 리듬**: 한국어 브랜드는 2~4음절이 최적 (삼성, 카카오, 네이버)
- **된소리 효과**: ㅃ,ㄲ,ㄸ는 강렬하지만 고급스러움과 거리
- **영문 호환**: 한글→영문 전환 시 발음과 표기가 자연스러워야 함

## 도메인/상표 사전 검토

### 도메인 확인 전략

| 우선순위 | 도메인 패턴 | 비고 |
|---------|-----------|------|
| 1순위 | brandname.com | 최상, 가격 높을 수 있음 |
| 2순위 | brandname.co / .io | 테크 브랜드에 적합 |
| 3순위 | getbrandname.com / trybrandname.com | 대안 |
| 4순위 | brandname.kr / .co.kr | 한국 시장 한정 |

### 상표 사전 검토 체크리스트

- [ ] KIPRIS (한국 특허정보원) 유사 상표 검색
- [ ] USPTO (미국) — 글로벌 진출 시
- [ ] 주요 SNS 계정명 사용 가능 여부
- [ ] 부정적 의미가 있는 언어가 없는지 (최소 영/중/일/스페인어)
- [ ] 경쟁사 이름과 혼동 가능성 없는지

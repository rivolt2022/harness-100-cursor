# Platform Algorithms — 플랫폼 알고리즘 방법론

sns-strategist와 visual-planner 에이전트가 콘텐츠 전략과 비주얼을 설계할 때 활용하는 알고리즘 전문 지식.

## 왜 알고리즘을 이해해야 하는가

좋은 콘텐츠를 만들어도 알고리즘에 의해 **10%의 팔로워에게만 노출**될 수 있다. 알고리즘은 적이 아니라 **활용해야 할 도구**다.

## Instagram 알고리즘 (2025 기준)

### 피드 랭킹 시그널

| 시그널 | 가중치 | 설명 |
|--------|--------|------|
| **관계** | 30% | 이전 상호작용 빈도 (좋아요, 댓글, DM, 태그) |
| **관심사** | 25% | 유사 콘텐츠 소비 이력 |
| **적시성** | 20% | 게시 후 경과 시간 (최근 우대) |
| **세션 빈도** | 15% | 자주 접속하는 유저에게 더 다양한 콘텐츠 |
| **팔로잉 수** | 10% | 팔로잉 많으면 개별 콘텐츠 노출 확률 감소 |

### 포맷별 알고리즘 차이

| 포맷 | 핵심 지표 | 최적화 전략 |
|------|---------|-----------|
| **피드 포스트** | 저장(Save) > 공유 > 댓글 > 좋아요 | 저장 가치 있는 정보형 콘텐츠 |
| **릴스** | 완주율 > 공유 > 리플레이 | 3초 내 훅, 15~30초 최적, 루프 구조 |
| **카루셀** | 스와이프율 > 저장 > 공유 | 첫 슬라이드 훅, 마지막에 CTA |
| **스토리** | 답장 > 투표/퀴즈 참여 > 탭 속도 | 인터랙티브 스티커, 질문 |

### Instagram 골든타임

| 대상 | 최적 게시 시간 |
|------|-------------|
| 일반 | 화~금 11:00~13:00, 19:00~21:00 |
| B2B | 화~목 08:00~09:00 |
| 주말 콘텐츠 | 토 10:00~12:00 |

## TikTok 알고리즘

### For You 페이지 랭킹

| 시그널 | 가중치 | 설명 |
|--------|--------|------|
| **완주율** | 35% | 영상을 끝까지 봤는가 (가장 중요) |
| **재시청** | 20% | 같은 영상을 반복 시청 |
| **공유** | 15% | DM/외부 공유 |
| **댓글** | 15% | 댓글 작성 및 길이 |
| **프로필 조회** | 10% | 영상 보고 프로필 방문 |
| **팔로우** | 5% | 영상 보고 팔로우 |

### TikTok 최적화 핵심

1. **첫 1초가 전부**: 스크롤을 멈추는 비주얼/텍스트 훅
2. **15~30초 최적**: 완주율 극대화 (긴 영상은 완주율 하락)
3. **루프 구조**: 끝이 시작으로 이어지면 재시청 유도
4. **트렌드 사운드**: 인기 음악/오디오 사용 시 노출 부스트
5. **텍스트 오버레이**: 무음 시청자를 위한 자막 필수

## LinkedIn 알고리즘

### 콘텐츠 분류 및 도달

| 분류 | 도달률 | 특징 |
|------|--------|------|
| **네이티브 텍스트** | 최고 | 외부 링크 없이 텍스트만 |
| **이미지 포스트** | 높음 | 1~3장 이미지 포함 |
| **문서/PDF** | 높음 | 슬라이드형 문서 업로드 |
| **비디오** | 중간 | 네이티브 업로드만 (YouTube 링크 X) |
| **외부 링크** | 낮음 | 외부 URL 포함 시 도달 40~50% 감소 |

### LinkedIn 골든 아워

게시 후 **첫 1시간의 반응**이 전체 도달을 결정한다:
- 첫 1시간에 댓글 10개+ → 알고리즘 부스트
- **전략**: 게시 직후 관련 인맥에게 DM으로 의견 요청

## X (Twitter) 알고리즘

### For You 피드 시그널

| 시그널 | 효과 |
|--------|------|
| **리트윗+인용** | 최고 가중치 — 확산 시그널 |
| **답글 스레드** | 대화 생성 = 콘텐츠 가치 |
| **북마크** | 2023년 이후 강화된 시그널 |
| **체류 시간** | 길게 읽는 콘텐츠 우대 |
| **이미지/비디오** | 텍스트만보다 도달 1.5~2배 |
| **외부 링크** | 도달 30~50% 감소 (셀프 리플라이로 분리) |

|------|-----------|------|
| **월** | LinkedIn | 주간 시작, 업무 모드 |

# IP Landscape Analysis — IP 랜드스케이프 분석

patent-mapper / protection-advisor 에이전트의 IP 전략 분석 역량 강화.

## IP 랜드스케이프 분석 프로세스

### 5단계 분석 절차

```
1단계: 기술 분야 정의
  - 기술 키워드 + IPC/CPC 코드
  - 검색식 설계

2단계: 데이터 수집
  - KIPRIS (한국), USPTO (미국), EPO (유럽)
  - 검색 범위: 최근 10~20년

3단계: 정량 분석
  - 출원 추이, 출원인 랭킹, 기술 분류
  - 특허 패밀리 분석

4단계: 정성 분석
  - 핵심 특허 선별, 청구항 분석
  - 공백 영역(White Space) 식별

5단계: 전략 도출
  - 기회/위협 매핑
  - 출원/매입/라이선스/회피 전략
```

## 특허 맵 유형

### 기술-시간 맵

```
         2020  2021  2022  2023  2024
기술 A    ██    ███   ████  █████ ██████
기술 B    █     ██    ███   ████  ████
기술 C    ███   ██    ██    █     █
기술 D    -     -     █     ██    ████

→ 기술 A: 급성장, 기술 D: 신흥
→ 기술 C: 쇠퇴
```

### 출원인 포트폴리오 맵

| 출원인 | 기술 A | 기술 B | 기술 C | 합계 |
|--------|--------|--------|--------|------|
| 삼성 | 45 | 32 | 18 | 95 |
| LG | 38 | 28 | 25 | 91 |
| 경쟁사 X | 52 | 15 | 8 | 75 |
| 우리 회사 | 12 | 20 | 5 | 37 |

### 인용 네트워크 맵

```
핵심 특허 식별 기준:
- 피인용 수 상위 10%
- 심사관 인용 (더 높은 가중치)
- 연결 중심성 (Betweenness Centrality)
```

## 공백 분석 (White Space)

### 기술-시장 매트릭스

```
         시장 A    시장 B    시장 C
기술 1    ████     ██       ░░░░  ← 공백
기술 2    ███      ░░░░     ██    ← 공백
기술 3    ░░░░     ███      ████  ← 공백

░░░░ = White Space (출원 부재/소수)
→ 전략적 출원 기회
```

### 공백 유형

| 유형 | 설명 | 전략 |
|------|------|------|
| 기술 공백 | 미개척 기술 영역 | 선제 출원 |
| 시장 공백 | 미적용 시장/지역 | 해외 출원 |
| 출원인 공백 | 경쟁사 미진입 영역 | 빠른 포트폴리오 구축 |

## FTO (Freedom to Operate) 분석

### FTO 분석 절차

```
1. 대상 제품/기술 명세 확인
2. 관련 특허 검색 (등록/유효 특허)
3. 청구항 분석 (구성요소 대비)
4. 침해 가능성 평가
   - 문언 침해: 모든 구성요소 일치
   - 균등 침해: 실질적으로 동일
5. FTO 의견서 작성
```

### 침해 리스크 매트릭스

| 리스크 | 침해 가능성 | 영향 | 대응 |
|--------|-----------|------|------|
| 🔴 높음 | 문언 침해 가능 | 핵심 기능 | 회피 설계/라이선스 |
| 🟡 중간 | 균등 침해 가능 | 일부 기능 | 회피 설계 검토 |
| 🟢 낮음 | 침해 가능성 낮음 | 비핵심 | 모니터링 |

## IP 전략 프레임워크

### 공격/방어 전략

| 전략 | 목적 | 수단 |
|------|------|------|
| 특허 울타리 | 핵심 기술 보호 | 주변 특허 출원 |
| 선점 출원 | 시장 진입 장벽 | 신기술 빠른 출원 |
| 크로스라이선스 | 상호 실시 | 포트폴리오 교환 |
| 방어 공개 | 경쟁사 출원 방지 | 기술 공개 (선행기술화) |
| 특허 풀 | 표준 기술 활용 | 업계 공동 라이선스 |

### IP 포트폴리오 최적화

```
유지 기준:
  ★★★ 핵심: 사업 직접 관련, 높은 가치
  ★★  중요: 방어적 가치, 잠재 수익
  ★   보통: 유지비 대비 가치 검토

정리 기준:
  - 사업 관련성 없음 + 라이선스 수요 없음 → 포기
  - 유지비 > 가치 → 매각 또는 포기
  - 만료 3년 이내 + 활용 계획 없음 → 포기 검토
```

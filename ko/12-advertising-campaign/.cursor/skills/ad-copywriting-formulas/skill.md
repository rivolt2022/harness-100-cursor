# Ad Copywriting Formulas — 광고 카피 설득 공식 라이브러리

copywriter 에이전트가 카피 작성 시 활용하는 검증된 설득 공식, 심리 트리거, 채널별 제약 조건 레퍼런스.

## 대상 에이전트

`copywriter` — 이 스킬의 공식과 패턴을 카피 작성에 직접 적용한다.

## 7대 설득 공식

### 1. AIDA (Attention → Interest → Desire → Action)
가장 보편적인 광고 카피 구조.

| 단계 | 목적 | 기법 | 예시 (피트니스 앱) |
|------|------|------|------------------|
| **A** Attention | 시선 탈취 | 충격 수치, 질문, 대비 | "하루 7분이면 충분합니다" |
| **I** Interest | 관심 유발 | 공감, 스토리, 데이터 | "헬스장 갈 시간이 없는 직장인을 위해 설계했습니다" |
| **D** Desire | 욕구 자극 | 혜택, 사회적 증거, 비전 | "12주 만에 평균 체지방 8% 감소" |
| **A** Action | 행동 촉구 | 긴급성, 한정, 무료 체험 | "지금 시작하면 30일 무료" |

**적합 채널**: 랜딩페이지, 긴 SNS 카피, 이메일, 영상 스크립트

### 2. PAS (Problem → Agitate → Solution)
고통 기반 설득. 전환 광고에 강력.

| 단계 | 목적 | 기법 | 예시 (회계 SaaS) |
|------|------|------|-----------------|
| **P** Problem | 문제 제시 | 타깃의 현재 고통 | "매달 영수증 정리에 12시간을 낭비하고 계신가요?" |
| **A** Agitate | 고통 증폭 | 미해결 시 결과 | "세무 시즌에 밤새 자료 찾다가 과태료까지..." |
| **S** Solution | 해결책 제시 | 제품 = 구원자 | "사진 한 장이면 자동 분류. 세금 신고까지 원클릭" |

**적합 채널**: 검색 광고, 리타겟팅 배너, 다이렉트 메일

### 3. BAB (Before → After → Bridge)
변화의 비전을 보여주는 구조.

| 단계 | 목적 | 기법 |
|------|------|------|
| **B** Before | 현재 상태 (불만) | "매일 퇴근 후 지쳐서 아무것도 못 하는 저녁" |
| **A** After | 이상 상태 (비전) | "에너지 넘치는 저녁, 취미와 가족과 함께하는 시간" |
| **B** Bridge | 연결 (제품) | "[제품명]이 그 다리가 됩니다" |

**적합 채널**: 브랜드 영상, SNS 스토리, 인플루언서 카피

### 4. 4Ps (Promise → Picture → Proof → Push)

| 단계 | 역할 |
|------|------|
| **Promise** | 핵심 혜택 약속 — "3개월 안에 영어 회화 자신감" |
| **Picture** | 결과를 시각화 — "외국인 동료와 자연스럽게 농담을 주고받는 당신" |
| **Proof** | 증거 제시 — "수강생 87%가 OPIC IM2 이상 달성" |
| **Push** | 행동 유도 — "오늘 등록하면 첫 달 50% 할인" |

### 5. SLAP (Stop → Look → Act → Purchase)
디지털 배너/숏폼 특화.

| 단계 | 글자수 | 역할 |
|------|--------|------|
| **S** Stop | 5자 이내 | 스크롤 멈춤 — 숫자/질문/대비 |
| **L** Look | 15자 이내 | 시선 유지 — 핵심 혜택 1개 |
| **A** Act | 10자 이내 | 관심 확인 — 클릭 유도 |
| **P** Purchase | CTA 버튼 | 전환 — "지금 시작" |

### 6. FAB (Feature → Advantage → Benefit)
기능 중심 제품의 카피 변환.

| 단계 | 설명 | 예시 (노이즈캔슬링 이어폰) |
|------|------|--------------------------|
| **F** Feature | 기능 사실 | "40dB 능동 소음 차단" |
| **A** Advantage | 기능의 의미 | "지하철에서도 스튜디오급 조용함" |
| **B** Benefit | 사용자 혜택 | "출퇴근길이 나만의 힐링 타임으로" |

### 7. Star-Chain-Hook
스토리텔링 기반.

| 단계 | 역할 |
|------|------|
| **Star** | 주인공(= 타깃 or 제품) 등장 |
| **Chain** | 연쇄적 사실/혜택으로 신뢰 구축 |
| **Hook** | 거절 불가능한 CTA |

## 심리적 트리거 패턴

| 트리거 | 메커니즘 | 카피 패턴 | 주의사항 |
|--------|---------|----------|---------|
| **긴급성** | 시간 제한 | "오늘 자정까지", "48시간 한정" | 허위 긴급성 금지 |
| **희소성** | 수량 제한 | "선착순 100명", "마감 임박" | 공정거래법 준수 |
| **사회적 증거** | 동조 효과 | "50만 명이 선택한", "평점 4.9" | 실제 수치만 사용 |
| **권위** | 전문가 인증 | "의사 95%가 추천", "특허 기술" | 출처 명시 필수 |
| **호혜성** | 선제 혜택 | "무료 체험 30일", "샘플 증정" | 조건 투명 공개 |
| **프레이밍** | 동일 사실의 관점 전환 | "98% 성공률" vs "2% 실패율" | 긍정 프레이밍 우선 |
| **앵커링** | 비교 기준점 | "원래 89,000원 → 오늘 39,000원" | 실제 정상가 기준 |

## 법적 주의사항 체크리스트

- [ ] "최고", "유일한", "제일" 등 최상급 표현 — 객관적 근거 필요
- [ ] "보장", "확정", "무조건" — 조건부 표현으로 대체
- [ ] "~효과", "~에 좋은" — 의약품/건강식품 표시광고법 확인
- [ ] 비교 광고 — 비교 대상/기준 명시 필수
- [ ] 할인율 표시 — 실제 판매가 기준, 허위 할인 금지
- [ ] "무료" 표기 — 부수 비용(배송비 등) 명시
- [ ] 타인 후기/수치 인용 — 출처와 기간 명시

# Media Mix Calculator — 미디어 지표 계산 & 예산 배분 모델

media-planner 에이전트가 미디어 플랜 수립 시 활용하는 핵심 지표 계산 공식, 예산 배분 프레임워크, 채널별 벤치마크 데이터.

## 대상 에이전트

`media-planner` — 이 스킬의 공식과 벤치마크를 미디어 플랜 수립에 직접 적용한다.

## 핵심 미디어 지표 계산 공식

### 도달 & 빈도 지표

| 지표 | 공식 | 설명 |
|------|------|------|
| **GRP** | `도달률(%) x 빈도` | 총 광고 노출량 |
| **TRP** | `타깃 도달률(%) x 빈도` | 타깃 대상 노출량 |
| **Reach** | `노출 순 사용자 / 타깃 모수 x 100` | 1회 이상 노출된 비율 |
| **Frequency** | `총 노출수 / 도달 사용자 수` | 1인당 평균 노출 횟수 |
| **Effective Frequency** | 일반적으로 3~7회 | 메시지 인지에 필요한 최소 노출 |

### 비용 효율 지표

| 지표 | 공식 | 해석 |
|------|------|------|
| **CPM** | `광고비 / 노출수 x 1,000` | 1,000회 노출 비용 |
| **CPC** | `광고비 / 클릭수` | 클릭 1회 비용 |
| **CTR** | `클릭수 / 노출수 x 100` | 노출 대비 클릭률 |
| **CPA** | `광고비 / 전환수` | 전환 1건 비용 |
| **CPL** | `광고비 / 리드 수` | 리드 1건 비용 |
| **CPI** | `광고비 / 설치 수` | 앱 설치 1건 비용 |

### 수익성 지표

| 지표 | 공식 | 해석 |
|------|------|------|
| **ROAS** | `광고 매출 / 광고비 x 100` | 광고비 대비 매출 (300%+ 목표) |
| **ROI** | `(매출 - 비용) / 비용 x 100` | 순수익 기준 수익률 |
| **LTV:CAC** | `고객 생애가치 / 고객획득비용` | 3:1 이상이 건전 |
| **Payback Period** | `CAC / 월 ARPU` | CAC 회수 기간 (개월) |

### 모델 2: 70-20-10 법칙

| 구분 | 비율 | 용도 |
|------|------|------|
| **검증 채널** | 70% | 과거 성과가 입증된 채널 |
| **확장 채널** | 20% | 성장 가능성이 있는 채널 |
| **실험 채널** | 10% | 새로운 채널/포맷 테스트 |

### 모델 3: SOV (Share of Voice) 기반

- SOV > SOM(시장점유율) → 시장점유율 성장 (Excess SOV 법칙)
- SOV가 SOM보다 10%p 높으면 → 연간 SOM 약 0.7%p 성장 (FMCG 기준)
- 신규 브랜드는 목표 SOM의 2배 SOV 권장

## 예산 시뮬레이션 공식

```
[입력] 총 예산 X원, 기간 N주, 목표 전환 Z건

일 예산 = X / (N x 7)
필요 클릭 = Z / 전환율(%)
필요 노출 = 필요 클릭 / CTR(%)
필요 CPM = 일 예산 / (일 필요 노출 / 1,000)

[검증]
실제 CPM <= 필요 CPM? → 실현 가능
실제 CPA x Z <= X? → 예산 충분
ROAS = (Z x AOV) / X x 100 → 목표 달성?
```

## 채널 시너지 매트릭스

| 조합 | 시너지 효과 | 권장 상황 |
|------|-----------|----------|
| 검색 + 디스플레이 | 디스플레이로 인지 → 검색으로 전환 | 퍼포먼스 캠페인 |
| SNS + 인플루언서 | 인플루언서가 신뢰 → SNS 광고가 전환 | 브랜드 론칭 |
| 영상 + 리타겟팅 | 영상으로 관심 → 배너로 재방문 유도 | 고관여 상품 |
| 검색 + SNS | 검색으로 의도 포착 → SNS로 관계 구축 | 서비스/SaaS |
| OOH + 디지털 | 오프라인 인지 → 온라인 전환 | O2O 비즈니스 |

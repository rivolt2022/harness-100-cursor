# Narrative Structure Patterns — 내러티브 구조 패턴 라이브러리

story-architect 에이전트가 비주얼 스토리 설계 시 활용하는 내러티브 구조, 감정 곡선, 비주얼 리듬 패턴.

## 대상 에이전트

`story-architect` — 이 스킬의 구조 패턴과 감정 곡선을 스토리 블루프린트 설계에 직접 적용한다.

## 내러티브 구조 패턴

### 1. 3막 구조 (Classic Three-Act)
비주얼 에세이의 가장 기본적인 구조.

| 막 | 비율 | 역할 | 비주얼 에세이 적용 |
|----|------|------|-----------------|
| **1막: 설정** | 25% | 세계관, 주제 도입 | 에스타블리싱 이미지 + 도입 텍스트 |
| **2막: 전개** | 50% | 갈등, 심화, 발견 | 장면 교차, 대비, 탐구 |
| **3막: 결말** | 25% | 통찰, 여운, 결론 | 클라이맥스 이미지 + 에필로그 |

### 2. 5장 구조 (기승전결확)
한국 전통 서사 + 비주얼 확장.

| 장 | 역할 | 이미지 톤 | 텍스트 톤 |
|----|------|----------|----------|
| **기** (도입) | 배경, 상황 | 와이드, 정적 | 서술적, 담담 |
| **승** (발전) | 심화, 확장 | 미디엄, 동적 | 관찰적, 디테일 |
| **전** (전환) | 반전, 발견 | 클로즈업, 대비 | 감정적, 질문 |
| **결** (결론) | 의미, 통찰 | 와이드 귀환 | 성찰적, 인용 |
| **확** (확장) | 독자 참여 | 열린 이미지 | 질문/초대 |

### 3. 영웅의 여정 (12단계 축약)
브랜드 스토리, 여행기, 성장 서사에 적합.

| 단계 | 이름 | 비주얼 에세이 적용 | 이미지 제안 |
|------|------|-----------------|-----------|
| 1 | 일상 세계 | 주인공의 현재 상태 | 평범한 일상 풍경 |
| 2 | 부름 | 변화의 계기 | 시선을 끄는 대비 |
| 3 | 거부 | 망설임 | 갈림길, 문턱 |
| 4 | 멘토 | 도움/영감 | 따뜻한 조명 |
| 5 | 문턱 넘기 | 여정 시작 | 문, 다리, 길 |
| 6~8 | 시련과 동맹 | 경험, 성장 | 다양한 장면 교차 |
| 9 | 클라이맥스 | 최대 도전 | 극적 조명/구도 |
| 10 | 보상 | 발견/성취 | 빛, 개방감 |
| 11 | 귀환 | 돌아옴 | 일상 세계 재방문 |
| 12 | 변화된 일상 | 성장한 시점 | 같지만 다른 풍경 |

### 4. 모자이크 구조 (비선형)
포토에세이, 도시 탐방, 감각적 에세이에 적합.

- 시간순이 아닌 **테마별 클러스터**로 구성
- 각 클러스터: 2~4장면, 하나의 감각/테마
- 클러스터 간 **시각적 모티프**로 연결 (반복 색상, 형태, 오브젝트)
- 독자가 자유롭게 의미를 구성하도록 열린 구조

### 5. 대비 구조 (Juxtaposition)
Before/After, 빛과 그림자, 두 세계의 비교.

| 패턴 | 구성 | 효과 |
|------|------|------|
| A-B-A-B | 두 주제를 번갈아 배치 | 대비의 리듬감 |
| A-A-A → B-B-B | 전반부 vs 후반부 | 전환점의 충격 |
| A+B 동시 | 분할 화면/오버레이 | 통합적 대비 |

## 감정 곡선 설계

### 감정 강도 패턴

#### 1. 상승형 (Ascending)
```
장면1 ━━ 장면2 ━━ 장면3 ━━ 장면4 ━━ 장면5
 20%    35%    55%    80%    100%
```
희망적, 성장 서사에 적합.

#### 2. 하강형 (Descending)
```
장면1 ━━ 장면2 ━━ 장면3 ━━ 장면4 ━━ 장면5
 90%    70%    50%    30%    15%
```
상실, 추억, 쇠퇴 서사.

#### 3. 산형 (Mountain)
```
        장면3
       ╱    ╲
 장면2        장면4
╱                ╲
장면1              장면5
```
클래식 서사 아크. 중앙에 클라이맥스.

#### 4. 파도형 (Wave)
```
  ╱╲    ╱╲    ╱╲
╱    ╲╱    ╲╱    ╲
```
리듬감 있는 에세이. 긴장과 이완의 반복.

#### 5. 계단형 (Staircase)
```
              ━━━
          ━━━╱
      ━━━╱
  ━━━╱
━━━╱
```
점진적 발견. 각 장면이 새 통찰 추가.

## 비주얼 리듬 (이미지-텍스트 배치)

### 밀도 패턴

| 패턴 | 이미지:텍스트 비율 | 효과 | 적합 |
|------|-----------------|------|------|
| **이미지 우세** | 70:30 | 시각적 몰입 | 풍경, 감각적 |
| **균형** | 50:50 | 정보 + 감성 | 다큐멘터리, 보고 |
| **텍스트 우세** | 30:70 | 사유, 깊이 | 철학적, 분석적 |

### 장면 내 배치 패턴

| 패턴 | 구성 | 리듬감 |
|------|------|--------|
| **풀블리드 → 텍스트** | 전면 이미지 다음에 텍스트 블록 | 호흡 - 사유 |
| **좌우 분할** | 이미지 좌 + 텍스트 우 (또는 반대) | 병렬 인식 |
| **텍스트 오버레이** | 이미지 위에 텍스트 | 통합적, 시적 |
| **캡션 하단** | 이미지 아래 짧은 캡션 | 사진 앨범 |
| **인용구 삽입** | 큰 따옴표 + 강조 텍스트 | 감정 앵커 |

## 장면 전환 기법

| 기법 | 설명 | 비주얼 적용 |
|------|------|-----------|
| **컷** | 즉각적 전환 | 장면 사이 여백 없음 |
| **페이드** | 점진적 전환 | 그라데이션 구분선, 여백 |
| **디졸브** | 겹침 전환 | 오버랩 이미지 |
| **모티프 매치** | 형태/색상 연결 | 비슷한 형태의 다른 대상 |
| **감정 매치** | 감정 톤 연결 | 같은 감정, 다른 장면 |
| **대비 컷** | 극적 차이로 전환 | 어둠→빛, 밀집→공허 |
| **내레이션 브릿지** | 텍스트가 장면 연결 | 전환 텍스트 삽입 |

## 장면 수 가이드

| 스토리 유형 | 최소 | 최적 | 최대 |
|-----------|------|------|------|
| 숏폼 에세이 | 3 | 5~6 | 8 |
| 표준 에세이 | 6 | 8~12 | 15 |
| 롱폼 스토리 | 10 | 15~20 | 25 |
| 포토에세이 | 8 | 12~16 | 20 |

### 장면당 권장 요소
- 이미지: 1~2장 (메인 + 보조)
- 텍스트: 50~150단어
- 캡션/인용: 10~30단어

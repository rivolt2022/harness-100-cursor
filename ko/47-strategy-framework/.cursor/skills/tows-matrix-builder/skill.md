# TOWS Matrix Builder — SWOT→전략 전환 방법론

swot-specialist와 strategy-writer의 전략 도출을 강화하는 스킬.

## 대상 에이전트

- **swot-specialist** — SWOT에서 TOWS 전략을 체계적으로 도출한다
- **strategy-writer** — TOWS 전략을 실행 로드맵으로 연결한다

## TOWS 매트릭스 구조

```
              | Strengths (내부 강점) | Weaknesses (내부 약점) |
──────────────┼─────────────────────┼──────────────────────┤
Opportunities | SO 전략              | WO 전략               |
(외부 기회)    | 강점으로 기회 활용    | 약점 보완하여 기회 활용 |
──────────────┼─────────────────────┼──────────────────────┤
Threats       | ST 전략              | WT 전략               |
(외부 위협)    | 강점으로 위협 대응    | 약점+위협 최소화       |
```

## 전략 유형별 도출 가이드

### SO 전략 (공격 전략) — "강점 × 기회"

```
질문: "이 강점을 활용하여 이 기회를 어떻게 극대화할 수 있는가?"

패턴:
1. 시장 확장: 기술력(S) + 성장 시장(O) → 신시장 진출
2. 신제품 개발: R&D 역량(S) + 고객 니즈(O) → 차별화 제품
3. 제휴/M&A: 브랜드(S) + 산업 재편(O) → 전략적 파트너십

예시:
  S: "AI 기술 개발 역량 상위"
  O: "헬스케어 AI 시장 연 25% 성장"
  SO: "AI 기술력을 헬스케어 진단 영역에 집중 투자하여 선점"
```

### WO 전략 (보완 전략) — "약점 × 기회"

```
질문: "이 약점을 보완하여 이 기회를 잡을 수 있는가?"

패턴:
1. 역량 강화: 인력 부족(W) + 시장 기회(O) → 채용/교육 투자
2. 파트너십: 채널 부재(W) + 수요 증가(O) → 유통 제휴
3. 외주/도입: 기술 부재(W) + 기술 성숙(O) → 솔루션 도입

예시:
  W: "글로벌 영업 네트워크 미비"
  O: "동남아 디지털 전환 가속화"
  WO: "현지 파트너사와 합작 법인 설립으로 동남아 진출"
```

### ST 전략 (방어 전략) — "강점 × 위협"

```
질문: "이 강점으로 이 위협을 어떻게 방어/전환할 수 있는가?"

패턴:
1. 차별화 강화: 기술력(S) + 경쟁 심화(T) → 기술 격차 확대
2. 전환 비용: 고객 관계(S) + 대체재(T) → 락인 강화
3. 선제 대응: 시장 지위(S) + 규제 변화(T) → 규제 주도

예시:
  S: "특허 15건 보유"
  T: "글로벌 대기업의 시장 진입"
  ST: "특허 포트폴리오를 활용한 기술 라이선싱 + 진입 장벽 강화"
```

### WT 전략 (회피/축소 전략) — "약점 × 위협"

```
질문: "약점과 위협의 교차점에서 피해를 어떻게 최소화하는가?"

패턴:
1. 축소/철수: 경쟁력 없는 사업 정리
2. 리스크 헤지: 의존도 분산
3. 구조 변화: 조직/프로세스 근본 개선

예시:
  W: "현금 흐름 불안정"
  T: "경기 침체 예상"
  WT: "비핵심 사업부 매각 + 현금 확보 + 핵심 역량 집중"
```

## 전략 우선순위 평가

```
각 TOWS 전략을 다음 기준으로 채점:

| 기준 | 가중치 | 1-5점 |
|------|--------|-------|
| 전략적 영향도 | 30% | 비즈니스 성과 기여 |
| 실행 가능성 | 25% | 현재 자원으로 실행 가능 |
| 시급성 | 20% | 지금 하지 않으면 기회 손실 |
| 리스크 | 15% | 실패 시 영향도 |
| 자원 효율성 | 10% | 투입 대비 효과 |

가중 합산 → 상위 3-5개 선택
```

## SWOT 요소 도출 가이드

### 내부 분석 (S/W) 프레임워크

```
7S Framework:
  Strategy — 전략적 포지션
  Structure — 조직 구조
  Systems — 프로세스/IT
  Shared Values — 문화/가치
  Style — 리더십
  Staff — 인력/역량
  Skills — 핵심 역량

VRIO 자원 분석:
  Valuable — 가치 있는 자원인가?
  Rare — 희소한가?
  Inimitable — 모방 어려운가?
  Organized — 활용 체계가 있는가?
  → 4가지 모두 Yes = 지속 가능한 경쟁 우위 (S)
  → No가 있으면 = 약점 또는 일시적 우위 (W)
```

### 외부 분석 (O/T) 프레임워크

```
PESTLE:
  Political — 정치/정책 변화
  Economic — 경제 상황
  Social — 사회/인구 변화
  Technological — 기술 발전
  Legal — 법률/규제
  Environmental — 환경/ESG

각 요소를 기회(O)와 위협(T)으로 분류
```

# Scoring Optimizer — 평가 배점 최적화 전문 도구

compliance-checker, plan-writer 에이전트의 배점 최적화 역량을 강화하는 전문 스킬.

## 적용 대상 에이전트

- **compliance-checker** — 배점 분석, 탈락 위험 평가
- **plan-writer** — 고배점 항목 중심 서술 전략

## 평가 배점 해부 프레임워크

### 공통 평가 구조 (정부 지원사업)

| 평가 영역 | 일반 배점 | 핵심 포인트 |
|----------|---------|------------|
| 기술성 | 30-40점 | 기술 차별성, 혁신성, IP 보유 |
| 사업성 | 20-30점 | 시장 규모, 매출 계획, 사업화 전략 |
| 수행역량 | 15-25점 | 대표자 경력, 인력, 인프라 |
| 사업계획 적절성 | 10-20점 | 추진 일정, 목표 구체성 |
| 정책 부합성 | 5-10점 | 정부 정책 방향 정렬 |
| 가점 | 최대 5-10점 | 여성/장애인 기업, 지역, 고용 |

### 배점 분석 4단계

```
1. 공고문에서 배점표 추출 → 항목별 점수 매핑
2. 고배점 항목 식별 → 상위 3개 항목 = 전체의 60-70%
3. 하위 항목 중 "안 쓰면 0점" 필수 항목 식별
4. 가점 항목 확인 → 쉽게 획득 가능한 가점 우선
```

## 고배점 항목별 서술 전략

### 기술성 (보통 최고배점)

| 평가 기준 | 고득점 전략 | 실패 패턴 |
|----------|-----------|----------|
| 기술 차별성 | 특허/논문 인용, 기존 기술 대비 정량 비교 | 추상적 "혁신적" 표현만 사용 |
| 기술 완성도 | TRL 단계 명시, 시제품/PoC 결과 제시 | 아이디어 수준에서 머무름 |
| 기술적 근거 | 수식/알고리즘/아키텍처 도면 포함 | 기술 설명 없이 기능만 나열 |

### 사업성

| 평가 기준 | 고득점 전략 | 실패 패턴 |
|----------|-----------|----------|
| 시장 규모 | TAM→SAM→SOM 단계적 산출, 출처 명시 | 근거 없는 시장 규모 주장 |
| 매출 계획 | Bottom-up 추정, 구체적 판매 채널 | Top-down만으로 추정 |
| 경쟁 분석 | 포지셔닝 매트릭스, 정량 비교표 | "경쟁자 없음" 주장 |

### 수행역량

| 평가 기준 | 고득점 전략 | 실패 패턴 |
|----------|-----------|----------|
| 대표자 역량 | 관련 경력 연차, 수상/인증 구체적 나열 | 이력 요약만 제시 |
| 인력 구성 | 핵심 인력 이름+학위+경력 명시 | "우수 인력 보유" 추상적 서술 |
| 인프라 | 장비, 시설, 파트너십 구체적 목록 | 일반적 설명 |

## 탈락 위험 체크리스트

### 자격 요건 (미충족 = 즉시 탈락)

- [ ] 기업 규모 기준 충족 (매출, 직원수, 자본금)
- [ ] 업종 제한 확인 (KSIC 코드)
- [ ] 설립 연한 요건
- [ ] 중복 수혜 제한 확인
- [ ] 체납 사항 없음
- [ ] 부실/부정 수급 이력 없음

### 형식 요건 (불비 = 탈락 또는 감점)

- [ ] 페이지 수 제한 준수
- [ ] 지정 양식 사용
- [ ] 필수 첨부 서류 완비
- [ ] 서명/날인/직인
- [ ] 제출 기한 준수

## 배점 시뮬레이션 도구

### 예상 점수 산출 매트릭스

```markdown
| 평가 항목 | 배점 | 자체 평가 | 예상 점수 | 개선 가능? | 개선 전략 |
|----------|------|----------|----------|-----------|----------|
| 기술 차별성 | 15 | 8/10 | 12 | 가능 | 특허 출원 증빙 추가 |
| 시장 분석 | 10 | 6/10 | 6 | 가능 | 시장 데이터 보강 |
| 대표자 역량 | 10 | 9/10 | 9 | 어려움 | - |
| 예산 적정성 | 5 | 7/10 | 3.5 | 가능 | 산출 근거 보강 |
| **합계** | **100** | | **73.5** | | 목표: 80+ |
```

### 개선 ROI 분석

노력 대비 점수 향상이 큰 항목부터 개선한다:
```
개선 ROI = (예상 점수 향상) / (소요 노력)
```

## 가점 획득 가이드

| 가점 항목 | 일반 점수 | 획득 난이도 | 준비 사항 |
|----------|----------|-----------|----------|
| 여성 기업 | 2-3점 | 자동 | 여성기업확인서 |
| 사회적 기업 | 2-3점 | 중간 | 인증서 |
| 고용 우수 | 1-3점 | 중간 | 고용증가 증빙 |
| 지역 소재 | 1-2점 | 자동 | 사업자등록증 |
| 녹색 기술 | 1-3점 | 높음 | 녹색기술인증 |

**전략: 자동 획득 가점은 반드시 챙기고, 중간 난이도 가점은 사전 준비한다.**

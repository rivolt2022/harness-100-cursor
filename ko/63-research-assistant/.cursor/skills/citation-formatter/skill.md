# Citation Formatter — 학술 인용 형식 변환 도구

reference-manager 에이전트의 서지 관리 역량을 강화하는 전문 스킬.

## 적용 대상 에이전트

- **reference-manager** — 참고문헌 형식 통일, 서지 정보 관리

## APA 7th Edition

### 본문 인용

| 상황 | 형식 | 예시 |
|------|------|------|
| 1저자 | (저자, 연도) | (Kim, 2023) |
| 2저자 | (저자1 & 저자2, 연도) | (Kim & Lee, 2023) |
| 3+저자 | (제1저자 et al., 연도) | (Kim et al., 2023) |
| 직접 인용 | (저자, 연도, p. 페이지) | (Kim, 2023, p. 45) |
| 40단어+ 인용 | 블록 인용 (들여쓰기) | 별도 단락 |
| 같은 저자 같은 연도 | (저자, 연도a) | (Kim, 2023a) |
| 단체 저자 첫 번째 | (전체 이름 [약칭], 연도) | (WHO, 2023) |

### 참고문헌 형식

**학술지 논문:**
```
저자. (연도). 논문 제목. 학술지명, 권(호), 페이지. https://doi.org/xxx
```

**단행본:**
```
저자. (연도). 서명 (판). 출판사.
저자. (연도). 서명 (편집자, Ed.). 출판사.
```

**편저 내 챕터:**
```
챕터 저자. (연도). 챕터 제목. 편저자 (Ed.), 서명 (pp. 페이지). 출판사.
```

**학위논문:**
```
저자. (연도). 논문 제목 [석사/박사학위논문, 대학명]. 데이터베이스/URL.
```

**웹페이지:**
```
저자/기관. (연도, 월 일). 제목. 사이트명. URL
```

**보고서:**
```
기관명. (연도). 보고서 제목 (보고서 번호). URL
```

## MLA 9th Edition

### 본문 인용

| 상황 | 형식 | 예시 |
|------|------|------|
| 기본 | (저자 페이지) | (Kim 45) |
| 2저자 | (저자1 and 저자2 페이지) | (Kim and Lee 45) |
| 3+저자 | (제1저자 et al. 페이지) | (Kim et al. 45) |

### Works Cited

```
학술지: 저자. "논문 제목." 학술지명, vol. 권, no. 호, 연도, pp. 페이지.
단행본: 저자. 서명. 출판사, 연도.
웹: 저자. "제목." 사이트명, 날짜, URL.
```

## Chicago (저자-날짜 방식)

### 본문 인용

```
(저자 연도, 페이지)
예: (Kim 2023, 45)
```

### 참고문헌

```
학술지: 저자. 연도. "논문 제목." 학술지명 권 (호): 페이지.
단행본: 저자. 연도. 서명. 도시: 출판사.
```

## BibTeX 변환

```bibtex
@article{kim2023,
  author  = {Kim, Minjun and Lee, Soojin},
  title   = {Article Title Here},
  journal = {Journal Name},
  year    = {2023},
  volume  = {15},
  number  = {3},
  pages   = {123--145},
  doi     = {10.1234/example}
}

@book{park2022,
  author    = {Park, Jihye},
  title     = {Book Title Here},
  publisher = {Publisher Name},
  year      = {2022},
  edition   = {2nd}
}
```

## 인용 품질 체크리스트

### 형식 검증

- [ ] 본문 인용과 참고문헌 목록 1:1 대응
- [ ] 본문에 있지만 참고문헌에 없는 항목 (orphan) 체크
- [ ] 참고문헌에 있지만 본문에서 인용되지 않은 항목 체크
- [ ] 동일 인용 스타일의 일관된 사용
- [ ] DOI/URL이 활성 상태인지 확인
- [ ] 저자명 표기 일관성 (Full name vs Initials)
- [ ] 연도 정확성

### 학술적 적절성

- [ ] 최근 5년 이내 문헌이 50% 이상인가?
- [ ] 피어리뷰 학술지 비율이 70% 이상인가?
- [ ] 1차 소스를 우선 인용했는가? (재인용 최소화)
- [ ] 핵심 문헌(고인용 논문)이 포함되었는가?
- [ ] 편향 없이 다양한 관점의 문헌이 포함되었는가?

## 한국어 문헌 인용 (APA 기준)

```
학술지:
김민준, 이수진. (2023). 논문 제목. 학술지명, 15(3), 123-145.

단행본:
박지혜. (2022). 서명. 출판사.

학위논문:
최영호. (2023). 논문 제목 [석사학위논문, 서울대학교].
```

**참고문헌 정렬 순서:** 한글 문헌(가나다순) → 영문 문헌(알파벳순) → 기타 언어

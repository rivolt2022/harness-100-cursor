# Personal Branding Harness

개인 브랜딩 에이전트 팀 하네스.

## 구조

```
.cursor/
├── agents/
│   ├── positioning-strategist.md
│   ├── resume-writer.md
│   ├── portfolio-designer.md
│   ├── profile-optimizer.md
│   └── cover-letter-writer.md
├── skills/
│   ├── personal-branding/
│   │   └── skill.md              — 오케스트레이터
│   ├── ats-optimizer/
│   │   └── skill.md              — ATS 통과 최적화 (키워드, 형식, 성과 정량화)
│   └── linkedin-seo/
│       └── skill.md              — LinkedIn SEO (검색 알고리즘, 섹션별 최적화, SSI)
└── CURSOR.md                     — 이 파일
```

## 사용법

Cursor 채팅에서 자연어 요청으로 실행하거나, `/personal-branding`로 수동 호출하거나, `@.cursor/skills/personal-branding/skill.md`를 컨텍스트로 첨부해 실행한다.

# LinkedIn SEO — LinkedIn 프로필 검색 최적화 도구

profile-optimizer 에이전트의 LinkedIn 최적화 역량을 강화하는 전문 스킬.

## 적용 대상 에이전트

- **profile-optimizer** — LinkedIn 프로필 최적화, 키워드 전략

## LinkedIn 검색 알고리즘 이해

### 검색 순위 결정 요인

| 요인 | 가중치 | 최적화 방법 |
|------|--------|-----------|
| 키워드 매칭 | 높음 | 타깃 키워드 전략적 배치 |
| 프로필 완성도 | 높음 | All-Star 레벨 달성 |
| 연결 네트워크 | 중간 | 500+ 커넥션 |
| 활동도 | 중간 | 주 1-2회 포스트/댓글 |
| 추천/보증 | 중간 | 5개+ 추천서, 기술 보증 |

### 키워드 배치 우선순위 (검색 가중치 순)

```
1. Headline (헤드라인) ← 가장 높은 가중치
2. Current Job Title (현재 직함)
3. About (소개) 섹션
4. Experience (경력) 설명
5. Skills (기술) 목록
6. Education (학력)
7. Recommendations (추천)
```

## 섹션별 최적화 가이드

### Headline (120자)

**공식:**
```
[타깃 직함] | [핵심 전문성] | [차별화 키워드]
```

| 유형 | 좋은 예 | 나쁜 예 |
|------|---------|---------|
| 개발자 | "Senior Backend Engineer \| Java & Kotlin \| MSA & Cloud Architecture" | "개발자" |
| 마케터 | "Growth Marketing Manager \| B2B SaaS \| SEO & Paid Acquisition" | "마케팅 담당자" |
| PM | "Product Manager \| Fintech \| Data-driven Product Strategy" | "PM at Company" |

### About 섹션 (2,600자)

**구조:**
```
[Hook] — 첫 3줄이 핵심 (더보기 클릭 전 노출)
  "N년간 [분야]에서 [핵심 성과]를 만들어온 [직함]입니다."

[Value Proposition] — 나만의 가치
  "저의 전문 분야:"
  • [전문성 1] + [정량 성과]
  • [전문성 2] + [정량 성과]
  • [전문성 3] + [정량 성과]

[Story] — 경력 서사 (선택)
  "[분야]에 입문한 계기부터 현재까지의 성장 스토리"

[Skills] — 키워드 리스트
  "핵심 기술: [키워드1], [키워드2], [키워드3]..."

[CTA] — 연락 유도
  "[관심사]에 대해 이야기 나누고 싶으시면 연락주세요."
```

### Experience 섹션

**각 경력 항목 구조:**
```
[직함] at [회사명]
[기간]

[1줄 역할 요약]

• [키워드 포함] [성과 1] — [정량 결과]
• [키워드 포함] [성과 2] — [정량 결과]  
• [키워드 포함] [성과 3] — [정량 결과]

기술: [관련 기술 키워드 나열]
```

### Skills 섹션

**전략:**
- 최대 50개 기술 등록 가능
- **상위 3개 기술** = 프로필에 노출되는 핵심 (타깃 키워드로 설정)
- JD에서 추출한 키워드와 정확히 일치시킨다
- 동료에게 기술 보증(Endorsement) 요청

### Featured 섹션

| 콘텐츠 유형 | 효과 | 권장 |
|-----------|------|------|
| 블로그 글 | 전문성 증명 | 2-3개 |
| 프로젝트 링크 | 실력 증명 | 1-2개 |
| 발표/컨퍼런스 | 리더십 증명 | 1-2개 |
| 미디어 인터뷰 | 영향력 증명 | 가능하면 |

## 리크루터 검색 패턴

### 리크루터가 사용하는 검색 쿼리

```
일반적 검색:
"Backend Engineer" AND "Java" AND "Spring Boot" AND "Seoul"

고급 검색 (LinkedIn Recruiter):
Title: "Senior" AND ("Backend" OR "Server")
Skills: "Java" AND "AWS"
Location: "Seoul"
Years of Experience: 5-10
```

### 검색에 걸리기 위한 체크리스트

- [ ] 타깃 직함이 Headline에 정확히 포함되어 있는가?
- [ ] 핵심 기술 키워드가 Skills에 등록되어 있는가?
- [ ] 현재 위치(Location)가 정확한가?
- [ ] 산업(Industry)이 올바르게 설정되어 있는가?
- [ ] Open to Work 설정이 활성화되어 있는가? (비공개 가능)

# Permit Requirements DB — 업종별 인허가 요건 데이터베이스

주요 업종의 인허가 유형, 요건, 구비서류, 처리기간을 체계적으로 정리한 DB.

## 인허가 유형 분류

### 허가·인가·등록·신고 구분

| 유형 | 법적 성격 | 심사 강도 | 예시 |
|------|----------|----------|------|
| 허가 | 금지 해제 (재량) | 높음 | 음식점 영업허가, 건축허가 |
| 인가 | 보충행위 (기속) | 높음 | 법인설립인가, 합병인가 |
| 등록 | 요건 충족 시 수리 | 중간 | 통신판매업, 학원 등록 |
| 신고 | 도달 즉 효력 (자기완결) | 낮음 | 사업자등록, 공유숙박 신고 |
| 면허 | 자격 기반 허가 | 높음 | 의료기관 개설, 약국 |

## 주요 업종별 인허가 요건

### 음식점 (일반음식점 영업허가)

```
관할: 시·군·구청 보건위생과
근거: 식품위생법 제37조
처리기간: 10일

인적 요건:
  - 식품위생법상 결격사유 미해당
  - 위생교육 이수 (6시간, 사전)
  
물적 요건:
  - 조리장: 식품과 직접 접촉하는 시설(스테인리스)
  - 급수시설: 상수도 또는 먹는물 수질기준 적합
  - 화장실: 조리장과 분리
  - 환기시설: 조리 시 발생하는 연기·증기 배출

구비서류:
  1. 영업허가신청서 (별지 서식)
  2. 영업시설의 배치도(층별)
  3. 교육이수증 (위생교육)
  4. 수질검사성적서 (지하수 사용 시)
  5. 건축물대장 (용도 확인)
  6. 대표자 신분증
  수수료: 28,000원
```

### 통신판매업 (등록)

```
관할: 시·군·구청 (관할 구청)
근거: 전자상거래법 제12조
처리기간: 즉시~3일

요건:
  - 사업자등록 완료
  - 구매안전서비스 가입 (에스크로 또는 소비자피해보상보험)

구비서류:
  1. 통신판매업 신고서
  2. 사업자등록증 사본
  3. 구매안전서비스 이용 확인증
  수수료: 없음
```

### 화장품 제조업 (등록)

```
관할: 지방식품의약품안전청
근거: 화장품법 제3조
처리기간: 14일

인적 요건:
  - 화장품 제조 관련 자격 또는 경력
  - 품질관리 책임자 (이공계 학사+)

물적 요건:
  - 제조시설: 제조실, 보관실, 시험실
  - 품질관리: 시험검사 장비
  - CGMP 기준 충족 (권장)

구비서류:
  1. 화장품제조업 등록신청서
  2. 시설 명세서 + 배치도
  3. 품질관리 책임자 자격증명 서류
  4. 사업자등록증 사본
```

### 소프트웨어 사업자 (신고)

```
관할: 한국소프트웨어산업협회
근거: 소프트웨어산업진흥법 제24조
처리기간: 7일

요건:
  - 사업자등록 완료
  - SW 관련 사업 수행 의사

구비서류:
  1. SW사업자 신고서
  2. 사업자등록증 사본
  3. 사업장 임대차계약서 (필요 시)
```

## 인허가 완전성 점검 체크리스트

```
□ 1단계: 인허가 유형 확인
  - [ ] 허가/등록/신고 중 어디에 해당하는가
  - [ ] 관할 기관 확인

□ 2단계: 인적 요건 점검
  - [ ] 결격사유 해당 여부
  - [ ] 필요 자격·경력 충족
  - [ ] 사전 교육 이수

□ 3단계: 물적 요건 점검
  - [ ] 시설 기준 충족
  - [ ] 설비·장비 구비
  - [ ] 면적 기준 충족

□ 4단계: 서류 완전성
  - [ ] 모든 구비서류 확보
  - [ ] 유효기간 내 서류 확인
  - [ ] 기재사항 정확성

□ 5단계: 접수 전 최종 확인
  - [ ] 수수료 준비
  - [ ] 접수 가능 일시 확인
  - [ ] 보정 가능성 대비
```

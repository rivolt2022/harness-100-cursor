# Vendor Negotiation Guide — 웨딩 업체 비교·협상 가이드

웨딩 업체별 가격 기준, 비교 포인트, 협상 전략을 제공한다.

## 주요 업체 유형별 가격 기준 (2024-2025)

### 웨딩홀

| 유형 | 가격 범위 (1인당) | 보증인원 | 특징 |
|------|-----------------|---------|------|
| 일반 웨딩홀 | 6-10만원 | 150-250명 | 가성비, 접근성 |
| 호텔 (4성) | 10-15만원 | 200-300명 | 서비스, 주차 |
| 호텔 (5성) | 15-30만원 | 200-400명 | 프리미엄 |
| 하우스웨딩 | 8-15만원 | 80-150명 | 독립 공간 |
| 야외웨딩 | 400-800만원 (대관) | 50-200명 | 자유도 높음 |
| 스몰웨딩 | 5-8만원 또는 정액 | 30-80명 | 소규모 |

**숨겨진 비용 체크리스트**:
```
□ 식대 외 VAT (10%) 별도 여부
□ 주차비 (무료 대수 확인)
□ 축의금 수수료
□ 추가 시간 사용료
□ 폐백실 사용료
□ 포토 존/야외 정원 사용료
□ 하객 봉투/식권 비용
□ 음향/조명 추가비
□ 보증인원 미달 시 페널티
```

### 스드메 (스튜디오·드레스·메이크업)

| 항목 | 가격 범위 | 포함 내용 | 주의사항 |
|------|----------|----------|---------|
| 스튜디오 | 80-300만원 | 원판 200-500장, 보정 30-80장 | 추가 보정 비용 확인 |
| 드레스 | 50-200만원 | 본식+촬영 2-3벌 | 액세서리 별도 여부 |
| 메이크업 | 40-100만원 | 리허설+본식 | 어머니 메이크업 별도 |
| 패키지 | 150-500만원 | 3종 통합 | 개별 합산 대비 10-20% 할인 |

**스드메 추가 비용 함정**:
```
□ 원판 추가 구매 (장당 1-3만원)
□ 보정 추가 (장당 3-5만원)
□ 드레스 그레이드업 (50-100만원+)
□ 액세서리 (티아라, 베일: 10-30만원)
□ 부케 (생화: 15-30만원)
□ 리허설 추가 촬영
□ 앨범/액자 업그레이드
□ 감독 촬영 (동영상: 50-150만원)
```

## 협상 전략

### 시기별 협상력

| 시기 | 협상력 | 전략 |
|------|--------|------|
| 비수기 (7-8월, 1-2월) | 높음 | 할인 15-20% 요청 가능 |
| 준성수기 (3-4월, 9-10월) | 중간 | 부가 서비스 요청 |
| 성수기 (5-6월, 11-12월) | 낮음 | 조기 예약 할인만 가능 |
| 일요일·공휴일 | 낮음 | 토요일 대비 할인 적음 |
| 토요일 오전 | 중간 | 오후 대비 10-15% 저렴 |

### 협상 포인트 TOP 10

```
1. "식대 단가 할인" — 하객 수로 협상 (200명+: 5-10% 할인 가능)
2. "보증인원 하향" — 실제 참석 예상에 맞춰 조정
3. "VAT 포함가" — 부가세 포함 총액 협상
4. "시간 연장" — 30분-1시간 무료 연장
5. "주차 추가" — 무료 주차 대수 확대
6. "폐백실 무료" — 별도 과금 면제
7. "원판 추가" — 스드메 원판 50-100장 추가
8. "드레스 그레이드업" — 상위 등급 드레스 포함
9. "리허설 메이크업 추가" — 무료 1회 추가
10. "결제 조건" — 분할납부, 카드 수수료 면제
```

### 협상 화법 템플릿

```
"다른 웨딩홀도 비교 중인데, [업체명]의 패키지가 마음에 들어서
먼저 여쭤봅니다. 식대 [X]만원에서 조정 가능할까요?"

"예산이 [X]만원인데, 이 범위 내에서
최대한 좋은 구성으로 제안해주실 수 있나요?"

"기본 구성은 좋은데, [추가 항목]을 포함해주시면
바로 계약하겠습니다."
```

## 계약 전 필수 확인 사항

```
□ 계약금·잔금 납부 일정 및 비율
□ 취소·변경 시 위약금 조건
□ 천재지변·전염병 시 대응 (코로나 사태 경험)
□ 포함 내역 상세 리스트 (서면)
□ 추가 비용 발생 가능 항목 (서면)
□ 하객 수 변경 가능 시점 (D-7 등)
□ 담당 플래너/코디네이터 배정
□ 리뷰 사진/영상 제공 범위
```

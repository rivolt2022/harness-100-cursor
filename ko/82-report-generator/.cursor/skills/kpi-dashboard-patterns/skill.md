# KPI Dashboard Patterns — KPI 대시보드 설계 패턴

analyst / executive-summarizer 에이전트의 지표 분석과 대시보드 구성을 표준화하는 패턴.

## KPI 정의 프레임워크 (SMART-R)

| 요소 | 설명 | 예시 |
|------|------|------|
| **S**pecific | 구체적 측정 대상 | "월간 신규 가입자 수" |
| **M**easurable | 정량적 측정 가능 | 명확한 수치 (건, 원, %) |
| **A**chievable | 달성 가능한 목표 | 전년 대비 +20% |
| **R**elevant | 비즈니스 목표와 연결 | 매출 → 고객 확보 → 가입자 |
| **T**ime-bound | 기간 명시 | 월간, 분기, 연간 |
| **R**esponsible | 책임자/부서 지정 | 마케팅팀 김팀장 |

## 대시보드 레이아웃 패턴

### 패턴 1: 경영진 대시보드 (Executive)

```
┌─────────────────────────────────────────┐
│  스코어카드 (핵심 KPI 4~6개)              │
│  [매출] [영업이익] [고객수] [NPS]          │
├──────────────────┬──────────────────────┤
│  매출 추이 차트    │  부문별 실적 차트      │
│  (꺾은선)         │  (누적 막대)          │
├──────────────────┼──────────────────────┤
│  비용 구조        │  리스크/이슈           │
│  (도넛)          │  (테이블)             │
└──────────────────┴──────────────────────┘
```

**원칙**:
- 위→아래: 요약→상세 (역피라미드)
- 스코어카드: 현재값 + 전기비 변화율 + 목표 대비 달성률
- 1페이지에 핵심만, 상세는 드릴다운

### 패턴 2: 운영 대시보드 (Operational)

```
┌─────────────────────────────────────────┐
│  실시간 상태 (트래픽, 에러율, 응답시간)     │
├──────────┬──────────┬───────────────────┤
│  시간별   │  일별     │  주간 트렌드       │
│  추이     │  비교     │                   │
├──────────┴──────────┴───────────────────┤
│  알림/이상 탐지 로그                       │
└─────────────────────────────────────────┘
```

## KPI 분석 방법론

### 분산 분석 (Variance Analysis)

```
실적 vs 목표 분산:
  분산 = 실적 - 목표
  분산율 = (실적 - 목표) / 목표 × 100

분산 등급:
  🟢 ±5% 이내: 정상
  🟡 ±5~15%: 주의
  🔴 15% 초과: 경고 → 원인 분석 필수
```

### 트렌드 분석

| 분석 유형 | 공식 | 용도 |
|----------|------|------|
| MoM (전월비) | (당월 - 전월) / 전월 × 100 | 단기 추세 |
| YoY (전년비) | (당월 - 전년 동월) / 전년 동월 × 100 | 계절성 제거 |
| CAGR | (최종값/초기값)^(1/n) - 1 | 장기 성장률 |
| 이동평균 | 최근 N기간 평균 | 변동성 완화 |

### 워터폴 분석 (Bridge Analysis)

```
전기 매출: 100억
  (+) 가격 인상 효과: +5억
  (+) 물량 증가: +12억
  (-) 환율 영향: -3억
  (-) 할인 증가: -2억
  (=) 당기 매출: 112억
```

브릿지 차트로 변동 요인별 기여도를 시각화.

## 스코어카드 설계

### 단일 KPI 스코어카드 구성

```
┌──────────────────────┐
│  매출                 │  ← 지표명
│  ₩145.2억            │  ← 현재값 (큰 폰트)
│  ▲ 12.3% vs 전월     │  ← 변화율 (색상 코딩)
│  목표 대비 96.8%      │  ← 달성률
│  ████████░░ 96.8%    │  ← 프로그레스 바
└──────────────────────┘
```

### 다중 KPI 비교

| KPI | 실적 | 목표 | 달성률 | 추세 |
|-----|------|------|--------|------|
| 매출 | 145.2억 | 150억 | 96.8% | ↗ |
| 영업이익률 | 12.3% | 15% | 82.0% | → |
| 신규고객 | 1,234명 | 1,000명 | 123.4% | ↗ |
| NPS | 72 | 75 | 96.0% | ↘ |

## 보고서 유형별 KPI 세트

### 월간 경영 보고

| 카테고리 | 핵심 KPI | 보조 KPI |
|---------|---------|---------|
| 재무 | 매출, 영업이익 | 매출원가율, EBITDA |
| 고객 | 신규고객, 이탈률 | CAC, LTV, NPS |
| 운영 | 생산성, 품질 | 가동률, 불량률 |
| 인사 | 이직률, 채용 | 교육시간, 만족도 |

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| KPI 정의 | SMART-R 6요소 충족 |
| 비교 기준 | 목표/전기/경쟁사 중 1개 이상 |
| 변화 방향 | 화살표/색상으로 즉시 파악 |
| 단위 통일 | 같은 차트 내 단위 일치 |
| 데이터 기준일 | 최신 데이터 날짜 명시 |
| 출처 | 데이터 소스 시스템 명시 |

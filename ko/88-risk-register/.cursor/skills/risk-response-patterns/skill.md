# Risk Response Patterns — 리스크 대응 전략 패턴

response-strategist / monitoring-planner 에이전트의 대응 계획 수립 역량 강화.

## 4대 대응 전략 (위협)

| 전략 | 설명 | 적합 조건 | 비용 |
|------|------|----------|------|
| **회피 (Avoid)** | 리스크 원인 제거 | 점수 ≥16, 제거 가능 | 높음 |
| **전가 (Transfer)** | 제3자에게 책임 이전 | 재무적 영향, 전문성 부족 | 중간 |
| **완화 (Mitigate)** | 확률/영향 감소 | 대부분의 리스크 | 중간 |
| **수용 (Accept)** | 리스크 감수 | 점수 ≤4, 비용 > 효과 | 낮음 |

### 전략별 상세 패턴

#### 회피 (Avoid)

```
패턴:
- 범위 축소: 리스크 높은 기능 제외
- 기술 변경: 검증된 기술로 대체
- 일정 연장: 충분한 시간 확보
- 벤더 변경: 신뢰성 높은 업체로

예시:
  리스크: "신기술 A의 안정성 미검증"
  회피: "검증된 기술 B로 대체, 기능 제한 수용"
  비용: 개발비 ₩3천만 추가, 기능 20% 축소
```

#### 전가 (Transfer)

```
패턴:
- 보험: 재해/사고 리스크
- 아웃소싱: 전문성 부족 영역
- 계약: 페널티/인센티브 조항
- 보증: 벤더 성능 보증

예시:
  리스크: "납기 지연으로 인한 위약금"
  전가: "하청업체와 납기 연동 위약금 계약"
  비용: 계약 단가 5% 상승
```

#### 완화 (Mitigate)

```
확률 감소 패턴:
- 사전 검증 (PoC, Pilot)
- 교육/훈련
- 리뷰/검수 프로세스
- 이중화/백업

영향 감소 패턴:
- 예비 자원 확보
- 대체 방안 준비
- 단계별 구현 (점진적)
- 모니터링 강화

예시:
  리스크: "핵심 인력 이탈" (확률 3, 영향 5 = 15)
  완화: "지식 공유 세션 + 백업 인력 지정"
  잔여: 확률 3, 영향 3 = 9 (🟡)
```

#### 수용 (Accept)

```
능동적 수용:
- 예비비 배정 (Contingency)
- 대체 계획 수립 (Plan B)
- 트리거 설정 + 대응 절차

수동적 수용:
- 발생 시 대처 (점수 ≤4)
- 비용 대비 대응 가치 낮음
```

## 리스크 대응 계획서 템플릿

```markdown
## 리스크 R01: [리스크명]

### 현재 평가
- 확률: [N] / 영향: [N] / 점수: [N×N] / 등급: [색상]

### 대응 전략: [회피/전가/완화/수용]

### 대응 조치
| # | 조치 | 담당 | 기한 | 비용 | 상태 |
|---|------|------|------|------|------|
| 1 | [구체 조치] | [이름] | [날짜] | [금액] | 진행중 |
| 2 | [구체 조치] | [이름] | [날짜] | [금액] | 미착수 |

### 잔여 리스크
- 대응 후 확률: [N] / 영향: [N] / 점수: [N×N]

### 트리거/조기 경보
- KRI: [지표] > [임계값] 시 활성화
- 모니터링 주기: [주간/월간]

### 대체 계획 (Plan B)
- [대응 실패 시 차선책]
```

## 리스크 모니터링 대시보드

### 리스크 히트맵 (현재 vs 목표)

```
현재 상태:
🔴 극심: 2건 (R01, R05)
🟠 높음: 3건 (R02, R07, R09)
🟡 보통: 5건
🟢 낮음: 8건

대응 후 목표:
🔴 극심: 0건
🟠 높음: 1건 (R05)
🟡 보통: 4건
🟢 낮음: 13건
```

### 트렌드 추적

| 리스크 | 전월 | 당월 | 추세 | 사유 |
|--------|------|------|------|------|
| R01 | 🔴 20 | 🟠 12 | ↓ 개선 | 완화 조치 효과 |
| R05 | 🟠 15 | 🔴 16 | ↑ 악화 | 신규 이슈 발견 |

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| 대응 전략 | 4대 전략 중 명시적 선택 |
| 조치 구체성 | 담당+기한+비용 명시 |
| 잔여 리스크 | 대응 후 재평가 수치 |
| 트리거 | KRI + 임계값 설정 |
| Plan B | 극심/높음 등급에 대체계획 |
| 비용 효과 | 대응 비용 < 리스크 EMV |

# Budget Planning — 이벤트 예산 계획

concept-planner / logistics-manager 에이전트의 예산 수립 역량 강화.

## 이벤트 예산 카테고리

### 비용 구조 (표준 배분)

| 카테고리 | 비율 | 세부 항목 |
|---------|------|----------|
| 장소 | 25~35% | 대관료, 부대시설, 보험 |
| 케이터링 | 20~30% | 식음료, 서빙, 장비 |
| 콘텐츠 | 15~20% | 연사비, 콘텐츠 제작, 장비 |
| 마케팅 | 10~15% | 홍보, 인쇄물, 디지털 |
| 운영 | 10~15% | 인력, 교통, 보안 |
| 예비비 | 5~10% | 불확실성 대비 |

### 예산 규모별 가이드

| 규모 | 인원 | 예산 범위 | 인당 비용 |
|------|------|----------|----------|
| 소규모 | ~50명 | ₩300~800만 | ₩6~16만 |
| 중규모 | 50~200명 | ₩800~3000만 | ₩8~15만 |
| 대규모 | 200~1000명 | ₩3000만~1.5억 | ₩10~15만 |
| 메가 | 1000명+ | ₩1.5억~ | ₩12~20만 |

## 예산 산출 템플릿

```markdown
# [이벤트명] 예산 계획서

## 기본 정보
- 일시: [날짜/시간]
- 장소: [장소명]
- 예상 인원: [N]명
- 총 예산: ₩[금액]

## 상세 예산

| 항목 | 단가 | 수량 | 소계 | 비고 |
|------|------|------|------|------|
| **[장소]** | | | **₩[소계]** | |
| 대관료 | ₩ | 1 | ₩ | |
| 장비 대여 | ₩ | N | ₩ | |
| **[케이터링]** | | | **₩[소계]** | |
| 식사 | ₩ | N명 | ₩ | |
| 음료/다과 | ₩ | N명 | ₩ | |
| **[콘텐츠]** | | | **₩[소계]** | |
| 연사 사례비 | ₩ | N명 | ₩ | |
| **총계** | | | **₩[총액]** | |
| 예비비 (10%) | | | ₩ | |
| **최종 합계** | | | **₩** | |
```

## 비용 절감 전략

| 전략 | 절감 효과 | 리스크 |
|------|----------|--------|
| 주중 개최 | 장소 30~50% | 참석률 감소 |
| 오전 시작 | 식사 1끼 절약 | 이동 불편 |
| 사내 장소 | 대관료 100% | 규모 제한 |
| 스폰서십 | 20~50% | 브랜딩 제약 |
| 하이브리드 | 장소·케이터링 축소 | 기술 비용 증가 |
| 조기 예약 | 10~20% | 유연성 감소 |

## 스폰서십 단가 가이드

| 등급 | 혜택 | 제안 금액 |
|------|------|----------|
| 타이틀 | 이벤트명 포함, 기조연설, 전면 노출 | 총 예산 30~50% |
| 플래티넘 | 부스+발표+로고 전면 | 총 예산 15~25% |
| 골드 | 부스+로고+인쇄물 | 총 예산 8~15% |
| 실버 | 로고+인쇄물 | 총 예산 3~8% |

## 품질 체크리스트

| 항목 | 기준 |
|------|------|
| 예비비 | 총 예산의 5~10% |
| 인당 비용 | 업계 기준 범위 내 |
| 고정/변동 | 구분 명시 |
| 결제 일정 | 계약금/중도금/잔금 |
| 취소 비용 | 취소 시 위약금 산출 |

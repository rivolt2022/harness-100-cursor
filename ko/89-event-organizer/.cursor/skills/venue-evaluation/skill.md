## 현장 답사 체크리스트

### 필수 확인 항목

- [ ] 수용 인원 (최대/권장)
- [ ] 레이아웃 변경 가능 여부
- [ ] AV 장비 (프로젝터, 스크린, 마이크, 스피커)
- [ ] WiFi (대역폭, 동시 접속 수)
- [ ] 전력 (콘센트 위치, 용량, 분전반)
- [ ] 조명 (조절 가능성, 자연광)
- [ ] 음향 (방음, 에코, 외부 소음)
- [ ] 케이터링 공간 (주방, 서빙 동선)
- [ ] 접근성 (엘리베이터, 장애인 시설)
- [ ] 주차 (대수, 요금, 발렛)
- [ ] 반입/반출 (하역장, 엘리베이터, 시간 제한)
- [ ] 대기 공간 (로비, 휴게실)
- [ ] 화장실 (개수, 위치, 상태)
- [ ] 비상구/소방 (안전 기준, 보험)

### 사진 촬영 필수 항목
1. 외관/입구
2. 메인 홀 (빈 상태)
3. AV 장비
4. 케이터링/주방
5. 화장실/편의시설
6. 주차장/접근 경로

## 계약 확인 사항

| 항목 | 확인 포인트 |
|------|-----------|
| 대관 시간 | 셋업/철수 시간 포함 여부 |
| 초과 비용 | 시간 초과, 인원 초과 단가 |
| 취소 정책 | 기간별 위약금 (30/60/90일) |
| 보험 | 장소 보험, 이벤트 보험 |
| 케이터링 제한 | 외부 업체 허용, 코르키지 |
| 음주 규정 | 주류 반입, 라이선스 |
| 장식 제한 | 벽 고정, 초 사용, 꽃잎 |
| 소음 제한 | 시간대별 데시벨 규제 |
